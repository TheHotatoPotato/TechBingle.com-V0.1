Best Gaming Keyboard?

The Razer ‘Huntsman’ starting at a $150. What’s different about this keyboard is that it has an ‘OPTO-MECHANICAL SWITCH’. A standard mechanical keyboard has a button at the bottom of the key which sends a signal to the computer after being pressed. But this keyboard has a beam of laser which when gets cut and sends a signal to the computer that a key has been pressed.
